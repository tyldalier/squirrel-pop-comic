This version of comic_git is obsolete. You can find the recommended version here, but read below for important info.

I've been working on a major revamp of comic_git for a while. This new version has a major overhaul in the underlying system for how the website is built, which gives the benefit that I can provide bug fixes and updates to existing users automatically. It also contains many small but needed improvements, like an improved default theme.

However, although that revamp is completely code-ready, and is already actively being used by at least two active webcomics, it is not yet "ready" for a full release. Mainly, the documentation has not been updated as needed to cover all the changes in the new version, due mostly to a lack of motivation on my part. Despite that, the documentation is still correct and useful enough that I feel comfortable recommending this "new" version of comic_git to anyone who wants to use it.

If you run into any issues using this "new" version, please don't hesitate to ask for help on our Discord server.

What you're seeing is the default website created by comic_git, a static-site generator hosted FOR FREE on GitHub Pages, that YOU can use to publish your own web comic easily and quickly!
